# OpenBlockCraft
A free software 3D block game inspired by Minecraft, Terasology, and the like, with a focus on performance and graphics.

### Supported Platforms

- Windows 7+
- MacOS 10.11+
- GNU/Linux (WIP)

OpenGL 3.3 (or higher) is required.
